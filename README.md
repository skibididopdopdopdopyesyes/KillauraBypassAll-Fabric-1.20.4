## Installation
1. Download latest release from Releases tab
2. Place in `.minecraft/mods/`
3. Download Fabric API for 1.20.4
4. Launch Minecraft with Fabric loader
5. Press **R** to toggle

## Configuration
Edit `config/killaura_bypass_all.json`
